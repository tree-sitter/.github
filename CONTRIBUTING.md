# Contributing

Thank you for your interest in contributing to the tree-sitter ecosystem.

## AI Usage Policy

The tree-sitter project has strict rules for AI usage. Please read the
full [AI Policy](https://github.com/tree-sitter/tree-sitter/blob/master/AI_POLICY.md)
before contributing. The short version:

- All AI usage must be disclosed with tool name and extent of assistance.
- You must fully understand every line of code you submit.
- No direct agent interaction (automated PRs, comments, code review).
- No LLM-generated PR descriptions or comments. Use your own words.
- Zero tolerance. Violations will be closed without discussion.
